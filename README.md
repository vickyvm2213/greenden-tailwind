# 🌿 Greenden - Tailwind CSS Project

Greenden is a simple responsive plant shop website built using **HTML** and **Tailwind CSS**.

## 🚀 Features

* 🏠 Home Page (Index Page)
* 🛍️ Product Page
* 📞 Contact Page
* 📱 Fully Responsive Design
* 🎨 Styled using Tailwind CSS

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* CSS3

## 📂 Project Structure

```
greenden/
│
├── index.html
├── product.html
├── contact.html
├── images/
└── README.md
```

## 📸 Pages Overview

### 🏠 Home Page

* Hero Section
* Featured Plants
* Call to Action

### 🛍️ Product Page

* Product Grid Layout
* Price Display
* Add to Cart Button (UI)

### 📞 Contact Page

* Contact Form
* Address Section
* Social Media Links

## 📱 Responsive Design

This project is fully responsive and works on:

* Mobile Devices
* Tablets
* Desktop Screens
